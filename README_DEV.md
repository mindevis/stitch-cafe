# Инструменты для разработки

## Установка инструментов качества кода

```bash
pip install -r requirements-dev.txt
```

## Использование инструментов

### Black (форматирование кода)
```bash
black .
```

### Ruff (быстрый линтер и форматтер)
```bash
ruff check .          # Проверка
ruff format .         # Форматирование
```

### MyPy (проверка типов)
```bash
mypy .
```

### Flake8 (линтер)
```bash
flake8 .
```

### Pylint (статический анализатор)
```bash
pylint .
```

## Логирование

Проект использует `loguru` для логирования:
- Логи сохраняются в папке `logs/`
- Ротация логов происходит каждый день в 00:00
- Хранятся логи за последние 30 дней
- Формат: `logs/bot_YYYY-MM-DD.log`

## Отладка

Для отладки можно использовать `icecream`:
```python
from icecream import ic

ic(variable)  # Выведет имя переменной и её значение
```
